  1. O que é HTML?

    2. Voltar

    HTML é uma linguagem de marcação, utilizada para criar páginas na internet. Mas o que é uma linguagem de marcação? É uma linguagem que opera por meio de tags (rótulos). Ela não tem comandos lógicos, como if-else, ou loops, como while e for. Com HTML, você monta o esqueleto de um site, utilizando rótulos para indicar parágrafos, títulos - em hierarquia ou não -, inserir imagens, links, listas etc.

  2. Principais TAGS

    3. Voltar

    As principais tags HTML são para texto, imagem, link e lista.

    1. Tags de texto

      2. Voltar

      As principais tags de texto são:

      • p: cria um parágrafo no corpo do texto html. Pode ser formatada por meio do comando align. Principais formatações: centralizado, justificado, alinhado à direita ou à esquerda.
      • h1: cria títulos. Vai de h1 a h6, sendo h6 o menor tamanho de título.
      • i, u, strong e mark: italico, subscrito, negrito e destacado, respectivamente.
      • small: deixa o texto pequeno.
      • sup: deixa o texto sobrescrito.
      • sub: deixa o texto subscrito.
      • blockquote:
        "destaca uma citação."
      • abbr: permite explicar uma sigla - EUA.

    2. Tags de imagem

      3. Voltar

      As principais tags de imagem são:

      • img: insere imagem no corpo do html. No atributo src, você deve colocar o endereço da imagem ou o nome do arquivo, caso ela esteja no mesmo diretório do seu projeto. No atributo alt, você deve representar a imagem textualmente. Caso o navegador não consiga abrir a imagem ou o usuário tenha deficiência visual, será possível ler a descrição da imagem.

    3. Tags de link

      4. Voltar

      As principais tags de link são:

      • a: permite que você insira links no seu html. Pode ser link interno, local ou externo. No atributo href, coloca-se o endereço da página. No atributo title, o título - irá aparecer quando você passar o mouse em cima. No atributo target, você define se a página será aberta na mesma ou em outra aba.
        • Link externo
        • Link interno
        • Local ou âncora

    4. Tags de lista

      5. Voltar

      As listas podem ser ordenadas e não ordenadas. Nas listas ordenadas, o HTML numera automaticamente cada item da lista. Na lista não ordenada, isso não acontece. Os elementos recebem apenas uma marcação de item no lugar do número.
